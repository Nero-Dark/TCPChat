Configure var "InstallDir" into "Install.bat" to your own installation path.

To install this service run "Install.bat" 2 times, first time in normal rights and second in admin rights.

You can configure parameters of the server by changing "Options.INI" before or after intallation, but you after intallation should change "Options.INI" along the path that you specified, then restart the service.
